# Contributing to Demarch

Thank you for your interest in contributing!

## Quick start

1. **Fork** the repo on GitHub
2. **Clone** your fork
3. **Branch** from `main`
4. **Open a PR** against `main`

Branch protection is enabled. Direct pushes to `main` are blocked for non-admins. PRs require at least one approving review.
